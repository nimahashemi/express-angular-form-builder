## 1. Requirments
- Node v18
- Angular v17
- mongodb v7

## 2. Run
- Go to the api directory -> npm i
- Go to the main directory -> node index.js
- Go to web/from-builder -> npm i
- Then run ng serve
- Backend run on port 3000 and fornt on 4200
- You can use http://localhost:4200 to run app in browser
